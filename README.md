# DDR-AI-Assignment
# AI DDR Report Generator

This project generates a Detailed Diagnostic Report (DDR) by combining data from:

- Inspection Report
- Thermal Report

## Features

- Extracts text from PDFs
- Uses LLM to analyze reports
- Generates structured DDR output
- Streamlit UI for easy interaction

## Setup

pip install -r requirements.txt

## Run

streamlit run app.py

## Output

Generated DDR report saved in outputs/final_report.md
